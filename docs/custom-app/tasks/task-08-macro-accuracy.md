# Task 08: Macro Accuracy Evaluator MVP

## Goal

Compare model predictions and gold labels in the simplest correct MVP form.

## Input Specs

- [json-output-schema.md](../specs/json-output-schema.md)
- [tsv-schema.md](../specs/tsv-schema.md)
- [evaluation-rule.md](../specs/evaluation-rule.md)

## Scope

- compare prediction pair(s) from JSON to TSV gold labels
- compute correct/incorrect result
- compute Macro Accuracy
- show evaluation result in the compact UI

## Completion Criteria

- one inference result can be scored against loaded gold data
