# Task 12: Add Prompt Preset UI To Setup

## Goal

Expose prompt presets in the setup screen so the user can replace the current prompt with a predefined template.

## Scope

- add a prompt-preset selector to the setup screen
- populate the system prompt editor when a preset is selected
- persist the selected preset and resolved prompt text

## Required Specs

- [prompt-preset-spec.md](../specs/prompt-preset-spec.md)

## Completion Criteria

- the setup screen supports choosing `Model A`, `Model B`, or `Model C`
- choosing a preset updates the system prompt field immediately
